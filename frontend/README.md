# Frontend

React + TypeScript frontend for Bibliophiles Bazar built with Vite.

## Requirements

- Node 18+
- npm, yarn, or pnpm

## Setup

1. Install dependencies:

```bash
cd frontend
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open the app in your browser at the address printed by Vite.

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Lint

```bash
npm run lint
```

## Notes

- The frontend target API is the backend service running on port `8080` by default.
- If you need a proxy for local development, add it to `vite.config.ts` and update the backend URL accordingly.
